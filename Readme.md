## Features

- Classic brick breaker gameplay
- Paddle and ball physics
- Score tracking
- Win / game over states
- Simple keyboard controls

## Controls

- ⬅️ **Left Arrow** – Move paddle left  
- ➡️ **Right Arrow** – Move paddle right  
- ↵ **Enter** – Restart after game over / win  

## How to Run

1. Install **Java JDK 8+**.
2. Compile the sources:

   ```bash
   javac -d bin src/*.java

**Run the game:**

java -cp bin Main

Or open the project in your favorite IDE (IntelliJ, NetBeans, Eclipse) and run the Main class.
